## <a name="introduction">ğŸ¤– Introduction</a>

**Next Ventures** is a modern startup directory platform designed for speed, performance, and aesthetics. It serves as a launchpad for entrepreneurs to pitch their ideas to the world.

Key capabilities include viewing dynamic pitches, filtering by category, and a seamless authoring experience for submitting new ventures. The platform is built on **Next.js 16** (Partial Prerendering enabled), employing a robust **Sanity CMS** backend for real-time content management.

- **Database:** Sanity Content Lake
- **Monitoring:** Sentry
